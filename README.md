#SEI ProFootball!

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~


A web app designed  for you to leave your own reviews on your favorite nfl teams.

_____

Technologies Used
Javascript
Node.js
Express.js
MongoDB
Mongoose
OAuth
Passport.js

______

WireFrame link

https://wireframe.cc/pro/pp/3c775304a269367

______

As a user

* Able to log into site and find my team.
* Have my favorite team stored with my log in.
* Create and save comments with the team/teams selected.
